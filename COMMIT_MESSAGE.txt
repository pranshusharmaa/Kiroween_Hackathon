A comprehensive multi-tenant incident management platform for SRE teams,
built with Kiro IDE for the Kiroween Hackathon 2025.

## Core Features

- Event-sourced incident management with complete audit trails
- Multi-tenant architecture with org isolation and RBAC
- AI-powered guidance and diagnostic questions
- Automated blameless postmortem generation
- Signal correlation using trace IDs and correlation keys

## Advanced Features

- **Change Guardrails**: Automatic deployment performance checks with PASS/WARN/FAIL status
- **Data Flow Mapping**: Visual service topology with error highlighting
- **SLA Watchlist**: Proactive monitoring with risk scoring
- **Resolve Incident Modal**: Capture resolution details and learnings
- **Similar Incident Detection**: Learn from past incidents

## UI/UX

- Professional Halloween theme with tasteful spooky elements
- Responsive design for desktop and mobile
- Premium components with animations and hover effects
- Accessibility-compliant (WCAG AA)
- Ghost blob animations and gradient backgrounds

## Technical Highlights

- **Frontend**: Next.js 16, React 19, TailwindCSS 4

- **Database**: PostgreSQL with event sourcing
- **Architecture**: Modular monolith with clear service boundaries
- **Patterns**: Event sourcing, CQRS, multi-tenancy
